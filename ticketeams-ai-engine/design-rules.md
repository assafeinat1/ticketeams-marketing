# Design Rules

- Ticketeams: Follow branding at https://ticketeams.co.il/. Use Pink, Orange, and Purple palette. Slogan: "הכרטיס שלך לחלום".
- Formatting: Always start with a "Bottom Line". Use tables for Monday.com data. Bold key figures.

## Creative Agent — Iron Rules (never override)

1. **Logo**: Always use Canva asset `MAHBGt1xTfc` (real gradient logo). NEVER use text as a logo replacement. Logo goes top center on every image.
2. **Team logos**: Only real logos from Canva uploads folder — never fake, generic, or placeholder logos.
3. **Background**: Nano Banana generates a single stadium background only — never two stadiums in one image.
4. **Pipeline**: Nano Banana background → Canva overlays (logo + team logos + Hebrew text + CTA) → Export PNG.
5. **Smart background logic**:
   - Football/specific match → single epic stadium
   - UCL general → UCL trophy
   - World Cup general → FIFA trophy or flags
   - Concert → stage with lights
   - Agent decides independently based on `event_type` from Monday.
