# nano-banana-manager

MCP server providing tools to interface with the Nano Banana model for high-fidelity image generation from text descriptions (with accurate text rendering) and image editing/composition.

## Installation

```bash
pip install -r requirements.txt
```

## Configuration

1. Edit the `.env` file and set your API key:
   ```
   NANO_BANANA_API_KEY=your_actual_key_here
   ```
2. The `.gitignore` ensures `.env` is never committed to git.

## Tools

### `GenerateImage`

Create high-fidelity images from text descriptions, including accurate text rendering. Provide a detailed prompt and optional parameters to control the output.

| Parameter | Type |
|-----------|------|
| `prompt` | string |
| `width` | int |
| `height` | int |
| `num_inference_steps` | int |
| `seed` | int |
**Returns:** `str`

### `EditImage`

Modify an existing image or compose multiple images into one. Provide a source image path, an editing prompt, and optionally a second image for composition.

| Parameter | Type |
|-----------|------|
| `image_path` | string |
| `prompt` | string |
| `secondary_image_path` | string |
| `strength` | float |
**Returns:** `str`
